The Basic Lively Kernel Setup A Lively World is a web page. The words "the world" and "this web page" mean the same thing. The page requires some javascript files. These are loaded when the page is loaded. Some .js files are always required to make Lively work, and some are specific to the page you are viewing. The lively web page may be on a server, such as Lively-web.org, or it may be hosted by your own machine. Running locally has several advantages. You can continue to work when you have no internet connection. No system code updates will happen locally unless you ask for them. This gives you a stable base of code.
